# How to Run

To find the sum total of a set of players:
1. `pipenv shell`
2. `pipenv install`
3. `python aggregate_stats.py`
4. Type in a list of players

To get the the average stats of a set of players:
1. `pipenv shell`
2. `pipenv install`
3. `python average.py` 
4. Type in a list of players

Whenever you want to end your pipenv session, type: `exit`.
